# STI Planning

Maquette statique pour organiser le planning interne de STI Traductions et donner aux interprètes/traducteurs un accès simple à leurs missions.

## Pages incluses

- Portail interne: Accueil, Planning, Missions, Collaborateurs
- Portail collaborateur: Accueil, Planning, Missions

## Lancer la maquette

Ouvrir `index.html` dans un navigateur.

La maquette fonctionne sans serveur et sans installation de dépendances.
